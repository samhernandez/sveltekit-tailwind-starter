# Sveltekit/Tailwind Starter

Minimum boilerplate for Sveltekit with Tailwind CSS.

- Plain JavaScript (untyped)
- No ESLint
- No Prettier
- No Playwrite (testing)

After cloning the repo, run the following:

```bash
# adjust to taste
npm install
rm -rf .git
git init -b main
git add .
git commit -m "Initial commit"
```
